WooText is a plugin to edit WooCommerce product buttons, badges, and other texts easily using customizer. 

== Description ==

The WooText is a small lightweight plugin to edit or customize the WooCommerce default buttons, badges, and texts easily using the customizer. By installing it you will be able to change the "Add to Cart" button text and many other text to anything you want.



This plugin can help you to change:

* Single product button text.
* Grouped product button text.
* External/Affiliate product button text.
* Variable product button text.
* Shipping charge text.
* Place order button text.
* Sale badge text.
* In stock text.
* Out of stock text.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wootext` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Appearence -> Customize -> WooText screen to configure the plugin.


== Frequently Asked Questions ==

= Why my all button text are gone after activating wootext plugin? =

You will have to write them from Appearance > Customize > WooText, as the purpose of the plugin is to change the default text of WooCommerce.

= Whare the default product button text will be shown? =

The default product button text is genarally shown on home page products.

= Can I use different text for different types of products? =

Yes you can!

= Is it possible to change other texts of WooCommerce? =

Probably it's possible. If you want to customize any other text feel free to [let us know here.](https://broadbrander.com/contact) We will try to include them to this plugin.
